# The Thrifty Backpacker — Travel Agent SaaS Platform

CMPE 131 group project. A multi-tenant SaaS platform that lets travel agencies
help budget-conscious travelers plan trips within a strict total budget.

**Scenario:** Violet, a 21-year-old college student, is planning a 1-week trip
to London with a strict $1,500 budget. She needs the cheapest flight + hostel
combination plus free walking tours.

## Tech Stack
- **Backend:** FastAPI (Python 3.10+), SQLAlchemy, PostgreSQL
- **Frontend:** Vue.js 3 + Vite + Tailwind (Phase 3 — baseline provided)
- **External API:** Amadeus (Flight/Hotel/Activity)
- **Auth:** JWT

## Repo Layout

- `backend/` — FastAPI server (built in Phase 2)
- `frontend/` — Vue.js UI (added in Phase 3 from baseline repo)
- `docs/` — Phase 1 PDFs, ERD, architecture diagram
- `.github/` — PR template
